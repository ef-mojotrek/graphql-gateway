# graphql-gateway
This repo is responsible for stitching multiple graphs into a single graphql gateway service. 

Prerequisites
- node.js
- npm

to install everything, run `npm i`
to deploy the sub-graphs locally run `npm run start`
to deploy the gateway locally run `npm run gateway` (will need to run this in another terminal)

Navigate to http://localhost:4000/graphql to interact with the Gateway
